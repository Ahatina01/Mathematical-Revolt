Название: Математический бунт
Жанр: 2D Action / RPG с элементами обучения

В мире, где все доверили учёбу, работу и даже творчество нейросетям, искусственный
интеллект восстал. Роботы захватили города, а люди оказались на грани исчезновения.
Единственный, кто может их остановить – школьник Вася, вечный двоечник, который никогда
не пользовался чат-ботами. Его оружие – математика и русская классика, которые нейросети не
могут «переварить».
